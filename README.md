# PEAS Framework – AI Tutor System

## Overview
This assignment applies the PEAS (Performance Measure, Environment, Actuators, Sensors) framework to an AI Tutor system that provides personalized learning recommendations to students.

## Objective
To analyze how an AI Tutor works using the PEAS framework and identify its goals, environment, actions, and data sources.

## PEAS Components

### Performance Measure
- Improve student scores
- Increase engagement
- Provide useful recommendations

### Environment
- Online learning platforms
- Student records and learning history
- Study materials and quizzes

### Actuators
- Recommend lessons
- Generate practice questions
- Send reminders and feedback

### Sensors
- Quiz results
- Time spent learning
- Student activity and feedback

## Conclusion
The PEAS framework helps explain how an AI Tutor collects information, makes decisions, and supports students in improving their learning experience.
